# 更新日志

本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [0.4.0] - Unreleased

### 新增

- 深色与浅色主题切换。
- 播放器弹幕设置、快捷键面板、下一集提示和播放诊断。
- 搜索历史、结果分类、加载骨架屏和失败重试。
- 排行榜筛选、Top 3 展示和日历当日高亮。
- Windows GitHub Actions 质量检查。
- 支持中文项目路径、依赖校验与 GPL 文档打包的 Windows Release 构建脚本。

### 改进

- 以天空日记与动漫杂志语言重写首页、发现、追番、资料库、下载、搜索、
  详情和设置页面。
- 播放器控制条组件化，补齐 hover、键盘焦点和语义标签。
- 播放器选集改为默认收起的右侧抽屉，支持按钮和 `Esc` 关闭。
- 封面图片增加内存/磁盘缓存与失败占位。
- 统一 ViraColors 主题颜色与中文错误提示。

### 修复

- 修复部分视频有声音但无画面的首帧异常。
- 修复播放器状态重复初始化导致的 `LateInitializationError`。
- 修复主题启动加载与 Modular 注入未共享同一实例的问题。
- 修复首页视频图片无法刷新的缓存回归。

## [0.1.0] - 2026-06-18

- 建立 Flutter Windows 项目。
- 完成番剧发现、搜索、详情、播放、收藏、历史与插件基础能力。
