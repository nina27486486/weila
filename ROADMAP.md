# 薇拉开发路线图

路线图描述方向而不是发布日期。功能只有在通过静态分析、自动化测试和 Windows
Release 构建后，才会标记为完成。

## v0.4 开源准备版

- [x] 统一深色/浅色主题与 ViraColors 语义颜色
- [x] 首页、搜索、详情、排行榜、日历和设置页体验重构
- [x] 播放器自动恢复、线路切换、无画面诊断与加载状态
- [x] 播放器控制条、弹幕面板、快捷键和下一集提示组件化
- [x] 封面缓存、搜索历史、搜索防抖与骨架屏
- [x] README、贡献指南、安全策略和社区模板
- [x] Windows CI：分析、测试与 Release 构建
- [x] 完成 Release 启动、首页数据与真实视频连续播放回归
- [x] 生成包含许可证和版本说明的 Windows ZIP
- [x] 完成 ZIP 在独立目录中的解压与启动验证

## v0.5 稳定性版

- 扩充播放器、下载、插件解析与 Hive 数据迁移测试
- 增加多线路长时间播放与自动切集回归
- 在干净 Windows 10/11 设备上验证下载、解压与首次启动
- 增加可导出的诊断日志，并默认移除敏感 URL 参数
- 增加插件超时、限流、失效与内容类型异常的统一提示
- 优化大列表滚动、图片内存占用与应用冷启动
- 建立设置和追番数据的备份/恢复

## v0.6 社区预览版

- 提供可校验的 Windows 压缩包与版本更新说明
- 建立插件规则示例、验证器和兼容性文档
- 增加匿名崩溃报告的可选方案，默认关闭并明确征得同意
- 完善键盘导航、高对比度和屏幕阅读器支持
- 根据社区反馈确定同步、投屏或画中画的优先级

## 长期方向

- 多设备数据同步
- 更可靠的放送日历与更新提醒
- 字幕轨道和外部字幕管理
- 画中画与外部播放器
- 跨平台评估

## 不在计划内

- 托管或分发未获授权的视频内容
- 绕过 DRM、付费墙或访问控制
- 在仓库中内置私人 API 密钥或用户数据
