# 本季选片柔光玻璃杂志卡设计

## 目标

将首页“本季选片”的海报轨道从直角、细线分隔的基础样式升级为更明确的卡片式导航。新样式延续薇拉“天空日记 × 动漫杂志”视觉，并与刊头的 Liquid Glass 语言保持一致。

本次只调整 `PosterRail` 及其卡片的展示与交互，不修改首页数据源、排序、路由或打开番剧的回调逻辑。

## 视觉结构

- 每张海报使用约 16px 圆角的柔光玻璃底板。
- 卡片外层使用一条低对比高光描边，以及近景、远景两层柔和阴影。
- 封面继续占据主要面积，但顶部圆角与信息区之间形成清晰层次。
- 标题与类型、评分信息放在独立底板中，保留单行省略，避免长标题破坏轨道高度。
- 左上角编号改为圆角玻璃胶囊，使用天空蓝文字和半透明纸张背景。
- 取消当前奇偶卡片的轻微旋转，使整条轨道更像规整的杂志选片墙。
- 日间主题使用白色与天空蓝高光；夜间主题使用深色纸张与低亮度白色描边。

## 交互

- 鼠标悬停时卡片上浮 6px，阴影加深，边框转为天空蓝，封面轻微放大。
- 动画时长使用现有快速动效令牌，封面缩放保持克制，避免轨道滚动时产生晃动。
- 点击仍调用现有 `PosterRail.onOpen`。
- 卡片支持键盘焦点，并可用 Enter 或 Space 打开。
- 键盘焦点显示清晰的天空蓝外环，不只依赖颜色或阴影表达状态。
- 横向滚轮、鼠标拖动、触控板和触控滚动行为保持不变。
- 当 `MediaQuery.disableAnimations` 为真时，禁用上浮和封面缩放，只保留静态状态变化。

## 组件边界

- `PosterRailItem` 的只读展示接口保持不变。
- `PosterRail` 的 `items`、`onOpen` 和 `height` 公共接口保持不变。
- 视觉变化集中在 `_PosterRailCard`，不引入 Store、Service 或领域模型依赖。
- 不增加第三方依赖，不加载新的外部美术资源。

## 响应式与异常状态

- 保持现有固定卡宽与横向轨道模式，避免桌面端卡片被压缩。
- 960、1280、1600 宽度下均保持可滚动且无布局溢出。
- 图片加载失败时继续使用 `CoverImage` 的现有占位规则。
- 空标题、空元信息和超长标题不改变卡片高度，也不阻断点击或键盘操作。

## 验证

- Widget 测试确认卡片具有圆角外壳、封面裁切层和玻璃编号胶囊。
- 交互测试确认悬停、键盘焦点、Enter/Space 打开及减少动态效果。
- 保留现有鼠标滚轮横向浏览测试。
- 首页在 960、1280、1600 宽度下运行无溢出。
- 完整执行 `flutter test`、`flutter analyze` 和 Windows Release 构建。

## 非目标

- 不修改本季片单内容、数量、排序或更新机制。
- 不把卡片改为全封面文字覆盖式布局。
- 不新增收藏、播放或更多菜单按钮。
- 不同步改造其他非 `PosterRail` 列表。
