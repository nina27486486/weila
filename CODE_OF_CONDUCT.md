# 社区行为准则

薇拉希望成为一个适合学习、讨论和协作的开源项目。参与 Issue、Pull Request、
Discussion 或其他项目空间时，请遵守以下规则。

## 我们鼓励

- 针对代码和行为讨论，不攻击个人。
- 给出可复现信息、技术依据和建设性替代方案。
- 尊重不同经验水平、语言背景和使用需求。
- 发现错误时帮助修正，而不是羞辱犯错的人。
- 尊重内容版权、隐私和第三方服务条款。

## 不可接受

- 骚扰、歧视、威胁、侮辱或持续的人身攻击。
- 发布他人的私人信息、凭据或未授权内容。
- 利用项目传播恶意软件、破解工具或违法内容。
- 刷屏、恶意引战或反复无视维护者的范围说明。

维护者可以编辑、隐藏或删除违反准则的内容，并限制严重或持续违规者参与项目。
如需私下报告社区问题，请通过 GitHub Security Advisory 联系维护者。
